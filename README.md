# Redis-like In-Memory Key-Value Store (C++)

This project is a simplified implementation of a Redis-like in-memory database built using C++ and socket programming.

## Features

* TCP server for client communication
* Key-value storage using hash maps
* Commands: SET, GET (DEL, TTL in progress)

## Tech Stack

* C++
* Socket Programming

## Status

Work in progress 🚧

## Goal

To understand how Redis works internally by building a simplified version.
